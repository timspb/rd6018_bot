- Репозиторий и хост синхронизированы по коммиту `c71b2594aced1cfacdc642f4583103f34ca77526`.

- AI-часть разделена:
  - кнопочный анализ: `ai_engine.py`
  - системные правила/промпт: `ai_system_prompt.py`

## Важные подтвержденные правила

- Глобальный лимит тока на всех этапах: `12.0A` (`MAX_STAGE_CURRENT`).
- OVP/OCP: `target + 0.1`.
- Для профилей Ca/Ca и EFB при тайм-ауте Main допустим принудительный переход в Mix.
- Дашборд должен работать как одно "рабочее" сообщение (редактирование/замена, без накопления дублей).
- `/help` не должен уходить в LLM/DeepSeek.
