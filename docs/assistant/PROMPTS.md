# Шаблоны Промптов Для Ассистента

## 1) Код-ревью логики заряда

Проведи code review по `charge_logic.py` и `bot.py`.
Фокус: переходы этапов, таймеры, safety-ограничения, регрессии.
Вывод: сначала критичные проблемы, потом средние, потом минорные.
Для каждого пункта дай файл, строку, риск и конкретный фикс.

## 2) Проверка синхронизации (локалка/GitHub/хост)

Проверь синхронизацию между локальным репо, origin/main и хостом `/root/rd6018_bot`.
Покажи три SHA и статус расхождений.
Если есть расхождения, выровняй до одного SHA и дай отчет.

## 3) Деплой и smoke-проверка

Подтяни свежий код на хост, перезапусти `rd6018-bot`, проверь статус сервиса
и последние логи. Затем выполни smoke-проверку ключевых команд:
`/start`, `/help`, `/modes`, `/off`, `AI`, `Логи`.

## 4) Актуализация документации

Сверь `README.md` и `RELEASES.md` с текущим кодом.
Обнови расхождения по лимитам, переходам, защите, командам и UX.
Сделай короткий список, что именно обновлено.
